 |                                 +---------------------------------------------------------------------------------------------+

 |                                 +---------------------------------------------------------------------------------------------+
 |                                 | `USB MIDI Device Class 1.0 <http://www.usb.org/developers/devclass_docs/midi10.pdf>`_       |
 +---------------------------------+---------------------------------------------------------------------------------------------+
 |   Audio                         |   I2S/TDM                                                                                   |

 +---------------------------------+---------------------------------------------------------------------------------------------+
